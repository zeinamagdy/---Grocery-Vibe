# 🛒 Grocery Vibe

A sleek, glassmorphic grocery list built with **React + Vite**. Designed for speed, persistence, and a smooth user experience.

## ✨ Features
- **Full CRUD**: Add, Read, Update, and Delete items.
- **Persistence**: Powered by `localStorage` (your list survives refreshes).
- **Glassmorphism UI**: Modern frosted-glass aesthetic with smooth transitions.
- **Mobile First**: Fully responsive for on-the-go shopping.

## 🚀 Tech Stack
- **React 18** (Functional Components & Hooks)
- **Vite** (Lightning-fast build tool)
- **CSS3** (Flexbox & Glassmorphism)

## 🛠️ Setup
1. `npm install`
2. `npm run dev`

## 🧠 Lessons Learned
- Managing complex state for "Edit Mode" (switching between view/input).
- Using the `useEffect` hook to sync state with `localStorage`.
- Handling dynamic CSS classes in React.
